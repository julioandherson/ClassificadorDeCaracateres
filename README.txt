CLASSIFICADOR DE CARACTERES

TÉCNICAS POSSÍVEIS:
*RANDOM SUBSPACE
*PART
*CLASSIFICATION VIA CLUSTERING
*NNGE

EXECUÇÃO:
1 - Certifique-se de que está dentro da pasta raiz do projeto (que contém o diretório src, o arquivo .jar e o arquivo script.sh).
2 - No terminal, execute: sh script.sh ARGUMENTO0 ARGUMENTO1 ARGUMENTO2 ARGUMENTO3

ARGUMENTOS*:
ARGUMENTO0: Verbose: Indica se os arquivos de teste serão impressos na saída padrão. Pode-se utilizar -v ou - para ativar ou desativar, respectivamente.
ARGUMENTO1: Tecnica Utilizada: São válidas as entradas: randomsubspace, part, classificationviaclustering e nnge.
ARGUMENTO2: Diretório com as imagens de treinamento: DEVE ser imagens/treinamento
ARGUMENTO3: Diretório com as imagens de teste: DEVE ser imagens/teste
